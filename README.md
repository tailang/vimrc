###### Useage
```bash
$rm -rf ~/.vim
$mkdir ~/.vim
$git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
```
Then clone the configuration
```bash
$git clone git@github.com:tailang/vimrc.git
$cd vimrc
$cp .vimrc ~/
```
Last open the new vim input the commond line 
```
:PluginInstall
````
